运动打卡系统

基于 Spring Boot + 微信小程序 + Vue3 管理后台的运动打卡系统。



系统功能：

普通用户功能（微信小程序端）
- 用户注册/登录（用户名+密码，JWT认证）
- 个人信息查看与编辑（昵称、头像、手机号、性别）
- 运动项目浏览（跑步、健走、骑行、游泳、瑜伽、篮球、羽毛球、健身）
- 每日运动打卡（选择运动、填写时长/距离/卡路里、提交打卡）
- 今日打卡状态查询
- 打卡历史记录查询（分页、日期筛选）
- 打卡日历视图（月历热力图，已打卡日期高亮）
- 个人运动数据统计（总打卡次数、连续天数、总时长、总卡路里）
- 运动趋势图表（近7天时长、运动类型分布）
- 积分系统（打卡+10分，连续打卡额外奖励）
- 成就徽章系统（10个成就，达成条件自动解锁）
- 打卡排行榜（本周/本月排行）

管理员功能（Vue3 Web后台）
- 管理员登录（角色校验）
- 仪表盘总览（核心指标卡片 + ECharts图表）
- 用户信息管理（列表查询、搜索、启用/禁用）
- 运动项目管理（增删改查）
- 打卡记录管理（查看、筛选、删除异常记录）
- 系统数据统计（打卡趋势、运动分布、用户增长图表）
- 积分记录管理（查看用户积分变动）
